# Bases_Unix_et_C

## Projet mini_teams

### Fichiers

- Makefile : makefile du projet
- client.c : script du client
- serveur.c : script du serveur

### Usage

- make : `clean` `fclean` `re` `.PHONY`
- client : `./client <PID cible> <Message>`
- serveur : `./serveur`

- Lancer le serveur
